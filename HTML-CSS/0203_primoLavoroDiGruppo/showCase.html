HTML

H1-H6

Permettono di formattare il testo. Hanno un corrispettivo semanticamente più forte.

Esempio

                <i>Testo corsivo</i>

Div

E' un generico contenitore e di tipo block-level element.
Ciò significa che ha dimensioni impostabili, larghezza che varia in base al contenitore e altezza generata dal contenuto.

Esempio

            
                <div>
                    <main>
                        <article>
                            <h4>Lorem</h4>
                            <p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>   
                        </article>   
                    </main>
                    <button>Cliccami</button>
                </div>

Span

E' un generico contenitore e di tipo inline element.
Ciò significa che ha dimensioni non impostabili se non con CSS, larghezza che varia in base al contenuto e altezza generata dal contenuto.

Esempio

            
                <span>Lorem ipsum dolor sit amet consectetur adipisicing elit.</span>

Link

Permette di collegare un file CSS all'html.

Esempio

            
                <link rel="stylesheet" href="file.css">

Style

Permette di scrivere in HTML il codice CSS. Non è utilizzato poichè preferibile l'SoC (Separation of contents)

Esempio

            
                <style>    
                </style>

CSS

Widht e Heigth

Valori rispettivamente della larghezza e dell'altezza dell'elemento (nel caso di testo del contenitore in cui è inserito). In caso di inserimento del padding, questo non viene aggiunto alle width e heigth specificate: il browser aggiungerà automaticamente altre W ed H a quelle presenti.

Esempio

div img {
width: 100px;
height: 200px;
}
